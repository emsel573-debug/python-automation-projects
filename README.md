# Wikipedia Content Scraper 🌐🐍

This is a Python-based automation tool that extracts structured data from Wikipedia articles. 

## 🚀 Features
- **Automatic Extraction:** Gets headings (H1, H2) and long paragraphs.
- **Clean Output:** Filters out short, irrelevant text.
- **File Saving:** Automatically saves the results into a `.txt` file.
- **Professional Logic:** Includes error handling for web connections.

## 🛠️ How to Use
1. Copy the code from `scraper.py`.
2. Change the `target_url` variable to any Wikipedia link you like.
3. Run the script and find your results in the `Download` folder.
